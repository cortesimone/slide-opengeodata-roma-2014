  2. Create Postgres User Role & DB
    (Using postgis template)
    CREATE ROLE osm LOGIN ENCRYPTED PASSWORD
'md5a6d3a895f1b5e97a65512a4a5d4878af'
  SUPERUSER CREATEDB
   VALID UNTIL 'infinity';

CREATE DATABASE osm
  WITH ENCODING='UTF8'
       TEMPLATE=template_postgis_20
       CONNECTION LIMIT=-1;
  3. Add hstore to DB
    CREATE EXTENSION HSTORE;

Routing

There are many offline, embedded and web-based routing services using OpenStreetMap data
Please visit: http://wiki.openstreetmap.org/wiki/Routing

  • pg_routing: extension for postgis with osm data support GPL,C++
  • OSRM C++,BSD, research oriented
  • Grasshoper Java, Apache, Scales from big server to mobile device, works on Raspberry Pi
  • OSRM C++,BSD, research oriented
  • Spatialite VirtualNetwork C, GPL/LGPL/MPL, portable

... and other services on cloud by Cloudmade, MapQuest, Skobbler...
